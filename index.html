This is Stratosphere.

Stratosphere is a completely OpenSource Virtual environment inspired by Proxmox VE, but made in Python.

Stratosphere is a fully open source community Virtual Environment made in Python. The goal is to create a modular Virtual Environment that can work by replacing Python scripts with any community-made script, private script, or custom-made script while being community-driven with plugins and is all about optimizing with the community, and opening full support with the community and from the developers, meanwhile while a Virtual Environment like Proxmox is not fully open source and still has enterprise support, it is paid for and i not meant for the community. our goal can be read more about here.
